School Database Management System


Developed a comprehensive School Database Management System for Delta Gemunupura Maha Vidyalaya, Gemunupura, Pussellawa, Sri Lanka. The mobile application was built using Android Studio as a hybrid solution, developed on Windows 11 using WSA (Windows Subsystem for Android), and implemented with Kotlin and Firebase. The system streamlines key school administrative processes, including student record management, attendance tracking, and examination result handling, significantly improving operational efficiency. This project received national recognition and was published in leading newspapers such as the Sunday Times and Sunday Observer.
